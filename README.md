# RoboVac

This project was developed by our group (4 members) as a submission to course project of COA Lab in 2nd year of our CSE Major. 

**It was a 4-part project**
  1. ISA Design
  2. Assembler Design
  3. Machine Simulation
  4. Micro Architecture Design
